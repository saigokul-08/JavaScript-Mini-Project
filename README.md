# JavaScript-Mini-Project
An interactive Simon Says memory game developed using Vanilla JavaScript, showcasing DOM manipulation, event handling, arrays, and game state management.

=> How to Play

1. Press any key on the keyboard to start the game.

2. The game will flash a random color â€” this is the first sequence.

3. Click the color button that was flashed.

4. With each correct input:

    -> The game moves to the next level

    -> A new color is added to the sequence

5. You must repeat the entire sequence in the correct order at every level.

6. If you click a wrong color:

    -> The game ends

    -> Your current level is displayed

7. Press any key again to restart the game and try to beat your score.
